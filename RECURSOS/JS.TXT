BIBLIOTECAS:

https://scrollmagic.io/  ScrollMagic: Es una biblioteca de JavaScript que permite crear efectos de animación y transición en la página web en función de la posición del usuario en la página.

https://scrollrevealjs.org/  ScrollReveal: Es otra biblioteca de JavaScript que permite crear efectos de animación y transición en la página web a medida que el usuario se desplaza hacia abajo en la página.

https://maxwellito.github.io/vivus/  Vivus.js: Es una biblioteca de JavaScript que permite crear animaciones de líneas vectoriales y gráficos en SVG de forma animada.

https://michalsnik.github.io/aos/  AOS: Es otra biblioteca de JavaScript que permite agregar efectos de animación a los elementos a medida que el usuario se desplaza hacia abajo en la página.
